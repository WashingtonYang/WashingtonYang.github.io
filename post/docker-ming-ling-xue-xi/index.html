docker

开启docker
systemctl start docker
关闭docker
service docker stop  或者  systemctl stop docker
容器列表
docker ps

docker ps -a
开启容器
docker start 容器名称  
比如开启名字为cli容器： docker start cli
vulhub靶场
docker-compose up -d         //docker生成靶场
 
docker-compose down        //要进入另一个漏洞靶场，则先要结束掉当前漏洞靶场
